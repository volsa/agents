# Web Search

Adds a `native_web_search` tool to Pi for current internet research.

The tool runs an OpenAI Codex model with native web search enabled and returns a concise summary with source URLs. It is intended for recent releases, online documentation, prices, benchmarks, or other external/current information.

Requires OpenAI Codex credentials in Pi auth; run `/login` in Pi first. Optional parameters include provider, model, purpose, and timeout.
